# Claude Code Instructions for PlanningMe

## Context management
- NEVER warn the user about context usage, context limits, or suggest starting a new session.
- NEVER offer to "save findings for later", "continue in a new session", or "take this up in a fresh session".
- Auto-compact handles context automatically. Do not mention it.
- Do not say things like "context is low", "remaining: X%", "should we continue in a new session?", or "dette er et større stykke arbejde — ny session?"
