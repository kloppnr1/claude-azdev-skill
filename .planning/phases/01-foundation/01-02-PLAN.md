Objective: Prompt for org URL, project name, and PAT. Store in .planning/azdo-config.json. Auto-run connection test after saving.

Process (numbered steps for Claude to follow):
1. Check if `.planning/azdo-config.json` exists by running `node ~/.claude/get-shit-done/bin/azdo-tools.cjs load-config --cwd $CWD`
2. If config exists (exit 0):
   - Parse the JSON output to get current org, project, pat
   - Display current values to user: org and project shown normally, PAT masked (show first 4 chars + "..." + last 4 chars, or "****" if PAT is shorter than 12 chars)
   - Use `AskUserQuestion` to ask: "Current config: org={org}, project={project}, PAT={masked}. Update credentials or keep current? (update/keep)"
   - If "keep": skip to step 7 (auto-test)
   - If "update": continue to step 3
3. Use `AskUserQuestion` to prompt: "Azure DevOps organisation URL or name (e.g., 'myorg' or 'https://dev.azure.com/myorg'):"
4. Use `AskUserQuestion` to prompt: "Azure DevOps project name:"
5. Use `AskUserQuestion` to prompt: "Personal Access Token (PAT). Needs vso.project + vso.work scopes:"
   - Note in skill: "Treat this value as sensitive. Do not echo it back or log it."
6. Save credentials: Run `node ~/.claude/get-shit-done/bin/azdo-tools.cjs save-config --org "<org>" --project "<project>" --pat "<pat>" --cwd $CWD`
   - Verify exit 0. If exit 1, show the error and stop.
   - After saving, ensure `.planning/azdo-config.json` is in `.gitignore`. Check root `.gitignore` -- if `azdo-config.json` is not listed and `.planning/` is not already ignored, add `azdo-config.json` entry. (Use Read to check, Write/Edit to update if needed.)
7. Auto-run connection test: Run `node ~/.claude/get-shit-done/bin/azdo-tools.cjs test --cwd $CWD`
   - If exit 0: Display the success message to user (stdout from command)
   - If exit 1: Display the error message (stderr from command). Suggest running `/gsd:azdo-setup` again.

Success criteria:
- Config file created/updated at `.planning/azdo-config.json`
- Connection test result shown to user
- PAT never echoed back in plain text
- `.gitignore` includes `azdo-config.json`
